# DerDieDas-Stellas-Deutsch
A premium, gamified German learning application with Russian translations, dynamic example sentences, and progress tracking. 

## Features
- **Vocabulary (Vokabeln)**: Categorized word lists with animated scenes, phonetic Cyrillic transcriptions, and context-rich examples. Features collapsible word groups.
- **Translator (Übersetzer)**: Auto-detecting DE/RU translator with quick-selection pills based on your vocabulary.
- **Games (Spiele)**: Interactive ways to test your knowledge including article matching (Tower Defense mode).
- **Statistics (Statistik)**: Track your learning progress over time with dynamic XP charts and accuracy donuts.

Hosted on GitHub Pages.
